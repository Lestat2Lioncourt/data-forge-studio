## Comment lire ce manuel

Chaque chapitre suit la meme structure :

1. **Situation** -- le besoin concret qui motive le scenario
2. **Pas a pas** -- les etapes detaillees pour y repondre
3. **Astuce** -- les raccourcis et bonnes pratiques pour gagner du temps
4. **Ce qui se passe en coulisse** -- pour comprendre le fonctionnement interne (optionnel)

Les chapitres sont independants : vous pouvez les lire dans l'ordre qui correspond a vos besoins du moment.

| Convention | Signification |

| F5 | Raccourci clavier |
